# Open WebUI

Open WebUI is a self-hosted web UI for chatting with LLMs.

## Verify

- Open `http://127.0.0.1:${PANEL_APP_PORT_HTTP}` after install.
- Set Ollama base URL to `http://ollama:11434` (default) if deployed in the same docker network.

## Rollback

- Re-install a previous version from the App Store (values will be replayed).
